# Cleaning Data Android

This project is used to clean and process iPhone data, then generate a final Excel file after cleaning.

## Project Structure

```text
cleaning data android/
│
├── data/
│   └── All data for iphone
│
├── datafinallllll.xlsx
│
└── README.md
```

## What This Project Does

The code will:

- Read the dataset file:

```text
All data for iphone
```

- The dataset is located inside the `data` folder.
- Clean and process the data.
- Generate a final Excel file named:

```text
datafinallllll.xlsx
```

## How to Use

1. Put the dataset file inside the `data` folder.
2. Run the code.
3. After execution, the final Excel file will be generated automatically.

## Output

Final generated file:

```text
datafinallllll.xlsx
```
